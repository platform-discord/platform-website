Platform's Commands



For these commands we assume that your prefix is the default (p?)

[] -> Optional

<> -> Required



General Commands

Command Description
p?randomcolour Gives a random colour in an embed.
p?colour <Hex> Generates an embed with a given Hex colour.
p?avatar [User] Sends yours or a given users avatar.
p?ping Gives the bots current ping.
p?info Gives some information about the bot.
